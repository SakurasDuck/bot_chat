# bot_chat

基于ChatGPT API 封装UI的聊天界面    

## 功能 :  

  #### 版本1 :  
  
   1. 通过ChatGPT API获取上下文聊天内容
   2. 配置聊天画像
   3. 设置自己OpenAI api key
   4. 设置代理(web 不能设置代理)  

  #### 版本2(进行中) :  

  1. 没有输入key时,高亮提示
  2. 抽象ui与聊天逻辑,实现插拔式聊天逻辑
  3. 实现多种聊天逻辑(对话,提问,图片生成,语音转文字)

## 支持  

1. Windows
2. Android
3. iOS
4. Web  

## 使用
> ui为手机样式,可能会适配多端
